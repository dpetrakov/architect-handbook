# AI Policy: [Название проекта]

## Разрешённые сценарии

| Сценарий | Класс инструмента | Ограничения |
|----------|-------------------|-------------|
| Генерация кода | AI-ассистент в IDE | Обязательный code review |
| Генерация тестов | AI-ассистент в IDE | Проверка покрытия и корректности |
| Анализ требований | Чат-ассистент (LLM) | Верификация аналитиком |
| Code review | Терминальный AI-агент | Дополняет, не заменяет ревью человеком |
| Генерация документации | Чат-ассистент (LLM) | Проверка архитектором |

## Запрещено

- Отправка production-данных, PII, секретов в AI без data processing agreement
- Автодеплой AI-сгенерированного кода без code review
- Использование AI для кода безопасности без экспертного ревью

## Верификация

- AI-сгенерированный код проходит обязательный code review
- AI-коммиты помечаются `Co-Authored-By: AI`
- Пересмотр политики: ежеквартально
